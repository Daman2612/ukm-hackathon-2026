**Description**: High-tech neon sci-fi hackathon registration website with vanilla JS, simulated backend, and 3D effects.
**Files**:
- index.html: Landing page with hero, countdown, and about sections. Refs: Tailwind, GSAP, Three.js, Lucide.
- register.html: Registration form with glassmorphism and validation. Refs: Tailwind, GSAP, Lucide.
- admin.html: Admin dashboard with mock login, data table, and charts. Refs: Tailwind, Chart.js, Lucide.
- faq.html: FAQ and simple static content page. Refs: Tailwind, Lucide.
- styles.css: Global custom CSS for neon glows, fonts, and animations.
- app.js: Global utilities, mock database using localStorage, and navigation logic.
- orb.js: Three.js logic for the abstract glowing digital sphere in the hero section.
